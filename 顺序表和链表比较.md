## 顺序表适合“多查少改”，链表适合“多改少查”
| 对比维度 | 顺序表（Array List） | 链表（Linked List） |
|---------|---------------------|---------------------|
| 存储方式 | 连续内存空间 | 非连续内存空间，通过指针连接 |
| 随机访问 | 支持，O(1) | 不支持，O(n) |
| 插入删除 | 中间操作需移动元素，O(n) | 修改指针即可，O(1)（已定位时） |
| 查找效率 | 按下标快 O(1)，按值需 O(n) | 只能顺序查找 O(n) |
| 空间利用 | 无额外指针，空间利用率高 | 需要额外指针域，占用空间 |
| 容量扩展 | 需要整体搬移扩容 | 动态分配，无需整体搬移 |
| 缓存友好 | 好，局部性强 | 差，节点分散 |
| 实现复杂度 | 简单 | 相对复杂 |
| 适用场景 | 查询多、修改少 | 修改多、查询少 |
